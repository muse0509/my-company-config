## CRITICAL 項目の詳細

### C1: Prediction トークン重複表示（10分）
```
問題: マーケット一覧で同じトークンが複数表示
原因: Token ID フィルタが重複を許容
修正: Array.prototype.filter で重複排除
テスト: マーケット 50+ で同一トークン なし を確認
```

### C2: WCAG AA アクセシビリティ違反（4-6h）
```
主な違反:
  - 色のコントラスト比不足（3:1 → 4.5:1 必要）
  - Alt テキスト欠落
  - フォーカス順序不正
  - ARIA ラベル不足

修正内容:
  - Lighthouse で 90+ スコア目指す
  - 色パレット調整
  - ARIA 属性追加
  - Tab order 修正
```

### C3: 入金フロー エラーハンドリング（2-3h）
```
問題: トランザクション失敗時のフィードバックが不十分
修正内容:
  - エラーメッセージの統一化
  - Retry ロジック追加
  - ユーザーに原因を明確に伝える
```

### C4: Virtualization 未実装（3-4h）
```
問題: 800+ トークン表示時に遅延
修正: React Window or TanStack Virtual 導入
期待: スクロール FPS 60+ 維持
```

### C5: ライトモード 未実装（6-8h）
```
実装内容:
  - カラートークン を light/dark で定義
  - Theme toggle UI 追加
  - CSS variables または Tailwind dark: class
  - ダークモード同等の品質保証
```
